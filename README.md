# LAB 4 – Application **FragmentsLab** 📱

## Cours
**Programmation Mobile Android avec Java**

## Objectif du TP
Ce laboratoire a pour objectif de découvrir l’utilisation des **fragments** dans une application Android.  
L’application permet de naviguer entre deux fragments à partir d’une activité principale.
